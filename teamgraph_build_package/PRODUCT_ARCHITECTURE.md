# PRODUCT_ARCHITECTURE.md

## System Overview

TeamGraph AI is the product and control layer around a Graphiti-powered live brain.

```txt
User / Admin
  -> Next.js Web App
  -> FastAPI Backend
  -> TeamGraph permission and safety layer
  -> Graphiti service wrapper
  -> Neo4j

External Agent
  -> @teamgraph/mcp CLI
  -> TeamGraph FastAPI backend
  -> API key validation
  -> Graphiti search / episode ingestion
  -> Neo4j
```

## Responsibilities

### TeamGraph

- authentication and demo session handling
- organization and project scoping
- admin/member permission enforcement
- approval queue routing
- audit and activity logging
- API key hashing and scope enforcement
- graph visualization normalization
- fallback retrieval when Graphiti is unavailable

### Graphiti

- live temporal memory ingestion
- episode-centric search
- LLM and embedding integration when configured

### Neo4j

- Graphiti storage backend
- TeamGraph metadata store for users, projects, approvals, keys, and activity

## Upload Flow

```txt
UI or CLI upload
  -> RawContext metadata written first
  -> Safety heuristics
  -> Curator classification
  -> safe          -> Context metadata + Graphiti episode ingest
  -> needs_review  -> ReviewItem only
  -> unsafe        -> Quarantine only
```

## Retrieval Flow

```txt
Brain Chat or MCP query
  -> TeamGraph validates role, project, key scope
  -> Graphiti search via service wrapper
  -> TeamGraph metadata enrichment
  -> Gemini answerer if configured
  -> deterministic fallback answer otherwise
```

## Safety Boundary

Never allow:

- raw Cypher generated by a model
- direct Graphiti access from the frontend or CLI
- unsafe or secret-bearing context into Graphiti
- member approval actions
- raw API key storage

Always route actions through backend services and validated TeamGraph schemas.

## Runtime Modes

- `live`: Graphiti initialized and serving retrieval/ingestion
- `fallback`: Graphiti unavailable or unconfigured, TeamGraph continues via Neo4j-backed deterministic retrieval

## Local Stack

- Next.js app on `localhost:3000`
- FastAPI API on `localhost:8000`
- Neo4j 5.26 via Docker Compose
- `@teamgraph/mcp` linked locally for agent access
